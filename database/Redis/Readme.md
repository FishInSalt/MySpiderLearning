## 3. 设置redis服务
![](https://github.com/FishInSalt/MySpiderLearning/blob/master/database/Redis/19.png)
## 4. 开启服务  
开启服务： redis-server --service-start  
另外，还有：
卸载服务： redis-server --service-uninstall
停止服务： redis-server --service-stop
